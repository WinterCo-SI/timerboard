Paste timers
Paste Discord timer lines (one per line). Region prefixes are parsed directly when present, e.g. 2026-04-09 16:29 [Geminate] D-I9HJ - Goonswarm Federation ihub [🛖IHUB][CONDI][Final] (1:42) [#Sov]. Non-region lines also work, e.g. 2026-02-16 13:00 KMC-WI - Drill P5M2 [Athanor][Friendly][Final] (70:59) [#Auth]
Timerboard quick start
Bring timers in from Discord, then use the table, compact, timeline, and map views to read them.
1
Copy from Discord
In Discord, go to #timerboard and copy the latest timer lines from the channel.
2
Select the timer block
Highlight the timer text you need, then use Ctrl+C to copy it.
3
Paste it here
Click Paste timers, click inside the box, use Ctrl+V, then choose Import & Replace for a fresh board or Import & Append to add to what is already here.
4
Read the pressure
Use filters for side, state, region, and structure type. The map view lets you click a region, then filtering will stay inside that region.
